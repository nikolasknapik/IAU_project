1.3 (A)

Hypotza 1 rozdiel v priemernej hodnote EtCO poda oximetry

Formulcia:

  • H: Priemern hodnota EtCO je rovnak pre merania s oximetry = 1 a oximetry = 0.
  • H: Priemern hodnota EtCO sa li medzi tmito dvomi skupinami.

Odvodnenie vberu:
Poda vsledkov korelanej analzy m EtCO (koncentrcia vydychovanho CO) najvyiu mieru vzahu s premennou oximetry. Preto sme overovali, i je tto zvislos aj tatisticky vznamn.

Vsledok (Welchov t-test):
Vpoet preukzal rozdiel medzi skupinami (p < 0.05), preto zamietame H.
Pri meraniach, kde bolo oximetry = 1, sa hodnota EtCO v priemere li (je mierne vyia) oproti meraniam bez oximetra.

Interpretcia:
Rozdiel naznauje, e aktivcia oximetra (oximetry = 1) je spojen s odlinmi podmienkami merania pravdepodobne s relnym pripojenm pacienta, ke sa zaznamenvaj vyie a stabilnejie hodnoty vydychovanho CO.

def cohen_d(x1, x2):
    nx1 = len(x1)
    nx2 = len(x2)
    s  = np.sqrt(((nx1-1)*np.std(x1, ddof=1)**2 + (nx2-1)*np.std(x2, ddof=1)**2) / (nx1+nx2-2))
    return (np.abs(np.mean(x1) - np.mean(x2))) / s

# rozdelenie podla oximetry na 1 a 0
g1 = df_obs['oximetry'] == 1
g0 = df_obs['oximetry'] == 0

def power_for(feature, alpha=0.05):
    x1 = df_obs.loc[g1, feature].dropna().to_numpy()
    x0 = df_obs.loc[g0, feature].dropna().to_numpy()
    d  = cohen_d(x1, x0)
    n1 = len(x1); n0 = len(x0)
    ratio = n0 / n1 if n1 > 0 else 1.0
    power = sm_stats.power.tt_ind_solve_power(effect_size=d, nobs1=n1, alpha=alpha, power=None, ratio=ratio)
    return {"feature": feature, "d": d, "power": power, "n1": n1, "n0": n0, "ratio": ratio}
